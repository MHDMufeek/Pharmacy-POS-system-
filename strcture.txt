Notes and highlights
- Backend uses Mongoose (MongoDB). Default dev MONGO_URI in `.env.example` points to local MongoDB.
- Auth is JWT-based; login returns { token, user } and the frontend stores the token in localStorage.
- Tests use `mongodb-memory-server` (no local DB required for tests).
- Frontend uses Vite; API base is centralized via `import.meta.env.VITE_API_BASE` (see `frontend/.env`).
